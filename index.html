¿Qué es HTML?

HTML (HyperText Markup Language) es un lenguaje de marcado de hipertexto que se utiliza para crear y estructurar páginas web:
Define el significado y la estructura del contenido de una página web Utiliza etiquetas y marcadores para definir cómo se presenta el contenido en un navegador Permite incluir texto, imágenes, videos, enlaces y otros recursos multimedia Tim Berners-Lee, uno de los inventores de la World Wide Web, creó el HTML en 1991. La primera versión era un diseño inicial de 18 elementos, de los cuales 13 aún se conservan.

Historia HTML

La historia del HTML (HyperText Markup Language) se remonta a 1980, cuando el físico Tim Berners-Lee propuso un sistema de hipertexto para compartir documentos en el CERN (Organización Europea para la Investigación Nuclear):

1991: Berners-Lee publica el primer documento formal sobre HTML, titulado “Etiquetas HTML”.
1993: Se reconoce la potencia de HTML para estandarizar los lenguajes de la Red.
1995: Se publica el HTML 2.0, el primer estándar oficial de HTML.
1997: Se publica HTML 4.0, que incluye novedades como las hojas de estilos CSS.
1999: Se publica HTML 4.01, la última versión oficial de HTML.
2006: Se publica HTML 5.0, la versión más reciente.

¿Que son Etiquetas en HTML?

Las etiquetas en HTML son elementos que se utilizan para estructurar y organizar el contenido de una página web. Se escriben entre signos de menor que (<) y mayor que (>), y generalmente vienen en pares: una etiqueta de apertura y una de cierre. La etiqueta de cierre incluye una barra inclinada (/) antes del nombre de la etiqueta.

¿Que son Atributos en HTML?

Los atributos en HTML son características adicionales que se usan dentro de las etiquetas para proporcionar información adicional sobre un elemento o para configurar su comportamiento. Los atributos siempre se colocan dentro de la etiqueta de apertura y consisten en un nombre y un valor, separados por un signo igual (=).

Reglas fundamentales en html

  1. Utilizar dos espacios para sangría.
  2. No dejar espacios en blanco al final.
  3. Usar la sintaxis HTML5.
  4. Incluir los atributos entre comillas dobles.
  5. Escribir las etiquetas en minúsculas para mayor coherencia y legibilidad.
  6. Incluir los atributos en la etiqueta de apertura de un elemento, no en la de cierre.
  7. Entrecomillar todos los valores de atributo para que el código sea más consistente y comprensible.

Etiquetas para hacer un documento html

  1. <!DOCTYPE html>: Define el tipo de documento
  2. <html>: Inicia el documento >
  3. <head>: Contiene información general sobre el documento, como el título, metaetiquetas, scripts y estilos
  4. <ttitle> </title>: Inserta el título del documento
  5. <body> </body>: Define el cuerpo del documento
  6. <div>: Define un bloque de contenido o sección de la página
  7. <a>: Crea enlaces a otros documentos, como páginas de internet, correos electrónicos o URL
  8. <strong>: Define texto importante, aplicando negrita en el navegador
  9. <br> Realiza un salto de línea
  10. <p>: Representa un párrafo
  11. <h1>…<h6>: Define títulos y subtítulos de la página
  12. <img>: Coloca una imagen en el documento
  13. <table>: Define una tabla
  14. <caption>: Define el título de la tabla
  15. <thead>: Define la cabecera de la tabla
  16. <tbody>: Define el cuerpo de la tabla
  17. <tfoot>: Define el pie de la tabla
  18. <tr> Define una fila en la tabla
  19. <th>: Define una celda de encabezado
  20. <td>: Define una celda de datos
