    >1-Atributos globales:

    class:

    En HTML, el atributo class se usa para asignar una o más clases a un elemento HTML. Las clases son útiles para aplicar estilos con CSS o para seleccionar elementos con JavaScript.

    id:

    En HTML, el atributo id se usa para asignar un identificador único a un elemento HTML. A diferencia de la class, que puede ser utilizada en varios elementos, un id debe ser único dentro de un documento HTML, lo que significa que no puede haber más de un elemento con el mismo id.

    lang:

    En HTML, el atributo lang se utiliza para especificar el idioma del contenido de un documento o de una parte específica de él. Esto es importante tanto para la accesibilidad como para los motores de búsqueda y herramientas de traducción automática.

    style:

    En HTML, el atributo style se utiliza para aplicar estilos CSS directamente a un elemento HTML de forma inline (en línea), es decir, dentro de la propia etiqueta HTML. El atributo style permite agregar reglas CSS específicas para un solo elemento, sin necesidad de usar una hoja de estilo externa o interna.

    tabindex

    El atributo tabindex en HTML se utiliza para controlar el orden de enfoque (focus) de los elementos interactivos de una página web cuando el usuario navega a través de ellos utilizando la tecla Tab del teclado. Es especialmente útil para mejorar la accesibilidad de las páginas web y para crear una navegación eficiente mediante el teclado.

    accesskey

    El atributo accesskey en HTML se utiliza para asignar una tecla de acceso directo a un elemento interactivo en una página web, de modo que el usuario pueda activar ese elemento usando una combinación de teclas sin necesidad de navegar hasta él con el ratón o el teclado. Esto es útil para mejorar la accesibilidad y facilitar la navegación para usuarios que prefieren o necesitan utilizar el teclado.

    ContentEditable

    El atributo contenteditable en HTML se utiliza para hacer que un elemento de la página sea editable por el usuario directamente en el navegador. Esto significa que el contenido del elemento puede ser modificado sin necesidad de acceder a un formulario o a un backend. El atributo puede ser aplicado a cualquier elemento HTML y convierte su contenido en editable de manera inmediata.

    contextmenu

    El atributo contextmenu en HTML se utiliza para asociar un menú contextual personalizado a un elemento. El menú contextual es el que aparece al hacer clic derecho (o presionar la tecla correspondiente en algunos dispositivos) sobre un elemento de una página web. Usualmente, este menú muestra opciones como "Copiar", "Pegar", "Inspeccionar", entre otras, dependiendo del navegador y de la página.

    data

    El atributo data-* en HTML se utiliza para almacenar información personalizada en elementos HTML, que no afecta al contenido visible ni al estilo de la página, pero permite que esa información esté disponible para usarla con JavaScript, sin necesidad de agregar atributos adicionales que no tengan una semántica estándar en HTML.

    2-Atributos form:

    name

    El atributo name en HTML se utiliza para asignar un nombre único a un elemento HTML, y su propósito varía según el tipo de elemento al que se aplique. Este atributo es particularmente importante en formularios HTML, donde se utiliza para identificar y enviar los datos del formulario al servidor.

    target

    El atributo name en HTML tiene varios usos importantes, pero su función principal es identificar un elemento dentro de un formulario para que, al enviarse, los datos asociados con ese elemento sean enviados al servidor con un nombre específico. A continuación, te explico sus principales usos y características.

    style

    El atributo style en HTML se utiliza para aplicar estilos CSS directamente a un elemento dentro del código HTML. A través del atributo style, puedes especificar propiedades de estilo (como color, tamaño, márgenes, etc.) directamente en el elemento, sin necesidad de usar un archivo externo o una sección <style> en el encabezado del documento.

    Action

    El atributo action en HTML se utiliza en el elemento <form> para especificar la URL a la que se enviarán los datos del formulario cuando este sea enviado. Es decir, el atributo action define la acción que ocurrirá cuando el usuario envíe el formulario, generalmente indicando el servidor o recurso que debe recibir y procesar los datos del formulario.

    Method

    El atributo method en HTML se utiliza en el elemento <form> para especificar el método HTTP que se usará al enviar los datos del formulario al servidor. El valor de este atributo determina cómo los datos del formulario se transmitirán al servidor (en la URL, en el cuerpo de la solicitud, etc.).

    Atributos input

    type

    El atributo type en HTML se utiliza para especificar el tipo de un elemento en función del contexto en que se utiliza, especialmente en formularios, donde se emplea para definir el tipo de un campo de entrada, botón, o control en general. Este atributo puede tener diferentes valores según el tipo de elemento al que se aplique.

    Name

    El atributo name en HTML se utiliza para asignar un nombre único a un elemento dentro de un formulario. Este nombre es clave porque permite que los datos del formulario sean identificados y enviados al servidor correctamente.

    Value

    El atributo value en HTML se utiliza para definir el valor de un campo de formulario o de un control en un formulario. Dependiendo del tipo de elemento en el que se utilice, el value puede representar el texto predeterminado, el valor de un botón, o el dato que se enviará al servidor cuando el formulario sea enviado.
