1-Etiquetas de documentos:

1.html

HTML (HyperText Markup Language) es un lenguaje de marcado utilizado para estructurar y presentar contenido en la web. Se usa para crear páginas web, y define la estructura básica de una página mediante el uso de etiquetas (o "tags") que indican cómo se debe mostrar el contenido.

2.head

En HTML, la etiqueta <head> es una parte fundamental de la estructura de una página web. Todo lo que se coloca dentro de la etiqueta <head> no se muestra directamente en la página web, pero proporciona información y recursos esenciales que afectan cómo se renderiza y comporta la página.

3.body

En HTML, la etiqueta <body> se utiliza para contener todo el contenido visible de la página web. Todo lo que se encuentra dentro de la etiqueta <body> es lo que los usuarios verán cuando visiten el sitio en su navegador, como texto, imágenes, enlaces, botones, formularios, tablas, etc.

2-Etiquetas de meta datos:

1.meta

En HTML, la etiqueta <meta> se utiliza para proporcionar metadatos sobre la página web, es decir, información adicional que describe o proporciona detalles sobre el contenido del documento. Estos metadatos no son visibles para el usuario en la página, pero son importantes para los navegadores web, motores de búsqueda, redes sociales y otros servicios.

2.style

En HTML, la etiqueta <style> se utiliza para definir reglas de estilo CSS Cascading aStyle Sheets directamente dentro de un documento ,aHTML. para decir, permite incluir código CSS en la propia página aHTML, en lugar de enlazar ha un archivo CSS externo.

3.base

4.link

En HTML, la etiqueta <link> se utiliza para vincular documentos externos con la página HTML. Generalmente, se usa para enlazar hojas de estilo CSS o recursos relacionados con la página web, pero también puede ser utilizada para otros fines, como especificar un ícono de la página o establecer relaciones con otros documentos.

3-Etiquetas de agrupacion:

1.p

En HTML, la etiqueta se utiliza para definir un párrafo de texto. Es uno de los elementos de bloque más básicos y comunes en HTML. Todo el contenido que va dentro de la etiqueta <p> se presenta como un bloque de texto con márgenes y saltos de línea automáticos antes y después del párrafo, lo que hace que el texto se muestre de manera organizada y legible.

2.hr

En HTML, la etiqueta <hr> se utiliza para crear una línea horizontal o separador visual en la página web. Esta línea se utiliza comúnmente para dividir o separar secciones de contenido dentro de una página, y ayuda a mejorar la estructura visual y la legibilidad del contenido

3.ul

En HTML, la etiqueta <ul> se utiliza para crear una lista no ordenada. Es un contenedor que agrupa elementos de lista, los cuales se representan con la etiqueta <li> (list item, o ítem de lista). Una lista no ordenada se muestra generalmente con viñetas (puntos) o círculos antes de cada elemento de la lista.

4.div

En HTML, la etiqueta <div> es un elemento de contenedor o bloque genérico que se utiliza para agrupar otros elementos HTML y estructurar el contenido de la página. Es uno de los elementos más versátiles de HTML, ya que no tiene un significado semántico por sí mismo, pero se usa para aplicar estilos, diseño, y organización del contenido de manera eficiente.

5.figure

La etiqueta <figure>en HTML se utiliza para representar contenido multimedia (como imágenes, gráficos, ilustraciones, diagramas, etc.) junto con su leyenda o descripción opcional, que normalmente se coloca dentro de una etiqueta <figcaption>. Esta etiqueta proporciona un contenedor semántico que mejora la accesibilidad y la comprensión del contenido, especialmente en términos de relación entre el contenido visual y su descripción o contexto.

6.figcaption

La etiqueta <figcaption> en HTML se utiliza para proporcionar una leyenda o pie de foto para el contenido dentro de un <figure></figure>

4-Etiquetas textuales:

1.a

La etiqueta <a> en HTML es utilizada para crear enlaces o hipervínculos que permiten a los usuarios navegar de una página web a otra, o incluso a una sección específica dentro de la misma página, o a otros recursos como archivos, correos electrónicos, etc.

2.spam

El spam es el término utilizado para describir el envío masivo e indiscriminado de mensajes no solicitados, generalmente con fines comerciales, promocionales o maliciosos. El spam puede presentarse en varios formatos, incluyendo correos electrónicos, mensajes de texto, publicaciones en foros, comentarios en redes sociales, y mensajes instantáneos. Los mensajes de spam suelen ser enviados a una gran cantidad de personas que no han dado su consentimiento para recibirlos.

3.sub

La etiqueta <sub> en HTML se utiliza para mostrar texto en subíndice. Esto significa que el texto que se coloca dentro de esta etiqueta se muestra más pequeño y alineado más abajo que el texto normal, lo que es útil en contextos como fórmulas químicas, matemáticas y otras notaciones científicas.

4.sup

La etiqueta <sup> en HTML se utiliza para mostrar texto en superíndice. El superíndice es un tipo de formato en el que el texto se muestra más pequeño y elevado respecto al texto normal. Esto es comúnmente usado en notaciones matemáticas, fórmulas científicas, exponentes, y otras representaciones que requieren elevar el texto.

5.cite

La etiqueta <cite> en HTML se utiliza para referenciar el título de una obra o una fuente, como un libro, una película, una canción, un artículo, una página web, una investigación, entre otros. Su propósito es indicar que el texto contenido dentro de la etiqueta es el título de una fuente citada, o bien el origen de una idea o contenido, especialmente en el contexto de citas académicas o de investigación.

5-Etiquetas semanticas:

1.nav

La etiqueta <nav> en HTML se utiliza para definir una sección de navegación dentro de una página web. Esta sección normalmente contiene enlaces (por ejemplo, menús, barras de navegación, enlaces a páginas relacionadas, etc.) que permiten a los usuarios moverse de una página a otra dentro de un sitio web o acceder a otras partes del mismo sitio.

2.section

La etiqueta <section> en HTML se utiliza para representar una sección genérica de contenido dentro de una página web. Este elemento define una parte del contenido que generalmente tiene un tema o propósito específico.

3.articule

La etiqueta <article> en HTML se utiliza para representar contenido autónomo y completo que puede ser distribuido de forma independiente, como artículos de un blog, publicaciones en redes sociales, noticias, entradas de un foro o cualquier otro tipo de contenido que tenga sentido por sí mismo fuera de la página donde aparece.

4.header

La etiqueta <header> en HTML se utiliza para representar el encabezado de una sección o página. Esta etiqueta generalmente contiene contenido introductorio, como el título del sitio web, el nombre de la página, logotipos, menús de navegación, o información relacionada con el encabezado de una sección dentro de la página.

5.main

La etiqueta <main> en HTML se utiliza para representar el contenido principal de un documento. Este contenido es el que está directamente relacionado con el propósito principal de la página web. En términos generales, todo lo que esté dentro de la etiqueta <main> es contenido único y esencial para la página, es decir, lo que le da sentido al propósito del sitio.

6.footer

La etiqueta <footer> en HTML se utiliza para representar el pie de página de un documento o de una sección específica dentro de una página web
